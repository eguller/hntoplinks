About HN Top Links

What is HN Top Links?

HN Top Links is a Hacker News aggregator that helps you discover the most engaging content by organizing stories based on different time periods. Similar to Reddit's "Top" feature, we group stories by:

  • Daily Top Stories
  • Weekly Best
  • Monthly Highlights
  • Yearly Favorites
  • All-Time Best

For each time period, you can sort stories by either most upvoted or most commented, helping you find the content that resonates most with the HN community.

Features

  • Browse top stories by different time periods
  • Sort by upvotes or comment count
  • Email subscriptions for daily, weekly, or monthly digests
  • Comprehensive archive access dating back to 2006
  • Clean, modern interface focused on readability

Open Source

HN Top Links is an open-source project. You can find the complete source code on GitHub. Feel free to contribute, report issues, or suggest improvements.
